01 Abstract

Protein Structure Analysis Tool

An interactive protein sequence analysis toolkit covering hydrophobicity, secondary-structure prediction, pairwise alignment, and physicochemical property estimation with SVG dashboards.

02 Highlights

Methods & contributions

  • Sequence biochemistry — monoisotopic and average residue masses, Kyte–Doolittle hydropathy, side-chain pKa/pI via Henderson–Hasselbalch.
  • Alignment — Needleman–Wunsch with affine (Gotoh) gaps scored on the full BLOSUM62 matrix.
  • Structure prediction — Chou–Fasman secondary-structure propensities and disorder prediction.
  • Visualization — SVG-based profiles served through a FastAPI dashboard.
03 Architecture

Data flow

sequence → hydropathy + composition
        → Chou–Fasman SS · disorder
        → Needleman–Wunsch (BLOSUM62) alignment → SVG
